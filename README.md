# Claude GTM Framework Docs

Public documentation and landing page for the **Claude GTM Framework**, a reproducible outbound system built on Claude Code agents, skills, and MCP integrations for Apollo, Instantly, Attio, and HeyReach.

## Live site

**https://amadejdemsar-create.github.io/claude-gtm-framework-docs/**

## What the framework is

See the full explainer at the URL above: what it is, how it fits together, the 7 agents and 8 slash commands, Day 1 walkthrough, how to onboard a teammate, costs, and troubleshooting.

## Framework source

The framework source (agents, skills, setup prompt) lives in the private repo `amadejdemsar-create/claude-gtm-framework`. Access is granted by invitation. If you are a collaborator and have accepted the invite, clone with:

```bash
git clone https://github.com/amadejdemsar-create/claude-gtm-framework.git
```

Then follow the setup instructions in `gtm-framework/setup-audit.md` inside that repo.
